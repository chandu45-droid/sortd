# V1 Product Scope — TaskFlow

**Status:** Defining (in progress)

## Concept
WhatsApp personal assistant for everyone. You text → it organizes your life. It reminds → you act.

## Target User
Anyone with WhatsApp. Students, professionals, freelancers, homemakers, founders — not role-specific.

## Use Cases
- Work: tasks, deadlines, project notes
- Fitness: gym schedule, workout reminders
- Social: movie plans, meetups, birthdays
- Career: interview prep, learning goals
- Personal: bills, errands, shopping lists
- Habits: daily reading, meditation, journaling

## Core Features
1. Natural language task creation via WhatsApp
2. Smart parsing (extracts deadline, priority, category from plain text)
3. Proactive: morning briefs, reminders, overdue nudges, daily wrap-ups
4. Notes/quick capture via chat
5. All notification timing user-customizable
6. Web dashboard for overview (secondary)

## Growth Path
Free personal → Pro features (paid) → Team/business mode (SaaS) → Full business OS
